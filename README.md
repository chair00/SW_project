3-1 소프트웨어설계 과목 프로젝트

QuickLink
: 긴 URL을 줄여서 보기쉽고 간결한 URL로 만드는 프로그램

- URL이 길면 다른 사람들에게 공유할 때 종종 불편함이 발생함.
- 이를 해결하기 위해 프로그램을 만듦

사용방법
- IN CMD, 'python QuickLink.py' 입력
- 생성된 주소로 들어가서 바꿀 url 입력창에 입력
- 제대로 동작한다면 짧은 url이 생성됨 (아니라면 invalid 메시지 뜸)
- copy 버튼을 눌러서 url 복사
- 웹브라우저에 복사한 url 넣어서 원래 url로 이동하는지 확인
